# Resumo de MarkDown

Aqui está um resumo de como se escrever em MarkDown

---

##  Títulos

**Como escrever:**
```markdown
# Título de Nível 1
## Título de Nível 2
### Título de Nível 3
````

**Como fica:**

# Título de Nível 1

## Título de Nível 2

### Título de Nível 3

-----

## 🔹 Estilização de Texto

**Como escrever:**

```markdown
**Este texto está em negrito**
*Este texto está em itálico*
~~Este texto está tachado~~
`Isso é um código em linha`
```

**Como fica:**

**Este texto está em negrito**
*Este texto está em itálico*
~~Este texto está tachado~~
`Isso é um código em linha`

-----

##  Listas

**Como escrever:**

```markdown
- Item de uma lista
- Outro item
  - Um sub-item do item anterior

1. Item numerado de uma lista
2. Outro item numerado
```

**Como fica:**

  - Item de uma lista
  - Outro item
      - Um sub-item do item anterior

1.  Item de uma lista numerada
2.  Outro item numerado

-----

##  Lista de Tarefas (Checklist)

**Como escrever:**

```markdown
- [x] Tarefa já concluída
- [ ] Tarefa pendente
```

**Como fica:**

  - [x] Tarefa já concluída
  - [ ] Tarefa pendente

-----

##  Links

**Como escrever:**

```markdown
[Este é um link para a documentação do GitHub](https://docs.github.com)
```

**Como fica:**

[Este é um link para a documentação do GitHub](https://docs.github.com)

##  Emojis

**Como escrever:**

```markdown
Você pode adicionar emojis digitando o código deles entre dois pontos. :rocket: :smile:

```

**Como fica:**

Você pode adicionar emojis digitando o código deles entre dois pontos. 🚀 😄

-----

##  Citações

**Como escrever:**

```markdown
> Esta é uma citação. Serve para destacar um texto
> que foi dito por outra pessoa ou de outra fonte.
```

**Como fica:**

> Esta é uma citação. Serve para destacar um texto
> que foi dito por outra pessoa ou de outra fonte.

-----

##  Bloco de Código

**Como escrever:**

````markdown
```javascript
function hello() {
  console.log("Olá, mundo do código!");
}
```
````

**Como fica:**

```javascript
function hello() {
  console.log("Olá, mundo do código!");
}
```

-----

##  Linhas Horizontais

**Como escrever:**

```markdown
Para criar uma linha divisória, use três ou mais traços em uma linha separada.

---
```

**Como fica:**

Para criar uma linha divisória, use três ou mais traços em uma linha separada.

##  Tabelas

**Como escrever:**

```markdown
| Cabeçalho 1 | Cabeçalho 2 |
|-------------|-------------|
| Célula 1.1  | Célula 1.2  |
| Célula 2.1  | Célula 2.2  |
```

**Como fica:**

| Cabeçalho 1 | Cabeçalho 2 |
|-------------|-------------|
| Célula 1.1  | Célula 1.2  |
| Célula 2.1  | Célula 2.2  |
